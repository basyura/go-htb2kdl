# README に --from の相対日付指定を追記

## 目的

`--from` に `yyyyMMdd` 形式だけでなく `-n` 形式の相対日付を指定できることを README に明記する。

## 修正案

- README の使い方またはオプション説明に、`--from -2` のような指定例を追加する。
- `-n` は実行日を基準に n 日前の 0:00 を開始日として扱うことを説明する。
- 無効な例や詳細な実装説明は増やしすぎず、既存 README の簡潔な文体に合わせる。

## 確認

- README の記述のみを変更するため、テスト実行は不要とする。

## 追加修正: completed の保持件数を README に追記

### 目的

`bookmarks.yml` の `completed` に保存される URL が最大 100 件であることを README に明記する。

### 修正案

- README の `bookmarks.yml` 説明付近に、`completed` は EPUB 生成済み URL の履歴であることを追記する。
- `completed` はユーザーごとに最新 100 件のみ保持し、100 件を超えると古い URL から削除されることを説明する。
- 保存期間による削除ではなく、件数による固定上限であることを短く説明する。

### 確認

- README の記述のみを変更するため、テスト実行は不要とする。
